# Portfolio Angga Dwi Kurniawan

Portfolio website built with Vue 3 + TypeScript + Tailwind CSS.

## Tech Stack
- Vue 3 (Composition API)
- TypeScript
- Tailwind CSS v4
- Vite

## Features
- Responsive design
- Dark theme with pink accent
- Smooth animations
- Loading splash screen
- Contact form with validation

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
